# Calculator

A fully functional console-based calculator focusing on simple and functional calculations. 

# Features
* Basic functions such as: addition, subtraction, multiplication and division.
* Further functions including: Square root, powers, and tenfolding.
* New History functions, allowing users to record, delete, and reuse calculations.

# How to use
Follow along with the prompts and enter the correct keys in order to take certain actions, the application will notify you if an option is invalid, minimizing confusion within the interface.
